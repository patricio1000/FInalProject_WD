# CampusConnect - Final Project

## 📌 Description
CampusConnect is a multi-page responsive web application that allows users to explore campus-related content such as events, dining, and resources. The Events page integrates live data from the TVMaze API to display dynamic content.

This project demonstrates front-end development skills including HTML, CSS, JavaScript, responsive design, and API integration.

---

## 🚀 Features
- Multi-page website (Home, Events, Resources)
- Dynamic data fetched from the TVMaze API
- Interactive buttons and event-driven JavaScript
- Responsive design for mobile, tablet, and desktop
- Clean and simple user interface
- Navigation bar with consistent layout across pages

---

## 🛠 Technologies Used
- HTML5 (semantic structure)
- CSS3 (custom styling + responsive design)
- Bootstrap 5 (layout and components)
- JavaScript (event-driven programming)
- TVMaze API
